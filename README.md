# blade-fm

该项目是基于`blade`框架搭建的音乐电台系统，使用七牛云作为存储CDN。
项目含括技术点：

+ `blade`框架使用
+ 模板引擎使用
+ 验证码技术
+ 第三方云存储使用
+ web音乐播放技术
+ 分片上传
+ 配置文件读取

### 使用

+ 建立`blade_fm`数据库，数据库文件在doc/blade_fm.sql
+ 将`blade-fm`部署在本地tomcat
+ 修改`blade.properties`配置文件

*注：*验证码库在`lib`目录下
